About the Project

This project uses applicant information and credit history data to predict credit card eligibility.

1

Data Collection

Application details and previous credit payment records are collected and merged.

2

Preprocessing

Missing values, duplicate records and categorical data are processed automatically.

3

Model Training

Multiple classification models are trained and compared using evaluation metrics.

4

Real-Time Result

The Flask application provides an instant approval or rejection prediction.

Technologies Used

The system is developed using popular data science and web-development technologies.

Python Pandas NumPy Scikit-learn XGBoost Flask HTML CSS Bootstrap

How the System Works

01

Enter Applicant Details

02

Machine Learning Analysis

View Prediction
